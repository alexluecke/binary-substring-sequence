# Challenge Problem:

Taken from https://leetcode.com/problems/count-binary-substrings/description/

# To Run:

With Typescript > 2.5.3 installed:

```
$ npm i
$ npm run solve -- --input <input>
```

Note:

```
<input> = <input> <code> | <code>
<code> = <code><code> | <terminal>
<terminal> = 0 | 1 | ''
```
